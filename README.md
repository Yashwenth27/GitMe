## 📌 Overview

Tired of the tedious process of manually gathering and uploading files, writing lengthy prompts, and crafting detailed README files from scratch? **GitMe** simplifies and automates this process using the power of LLMs.

This summer project demonstrates how generative AI can be used to streamline the README creation workflow for developers, making documentation fast, clean, and professional.

## 🚀 Features

✅ Provide just your **GitHub repo URL** — no manual uploads needed  
✅ Automatically fetches and analyzes key project files  
✅ Uses **Google’s Gemini LLM API** to understand your code deeply  
✅ Generates a **tailored README** with:  
- 🎯 Clear project overview  
- ⚙️ Installation guide  
- 🚀 Usage and Features  
- 🛠️ Tech stack badges  
- 🌱 Future enhancements  
- 🤝 Contribution guidelines  

## 🛠️ Technologies Used

- Python 🐍  
- Streamlit 🎈  
- Google Gemini LLM API 🧠  
- GitHub REST API  

## 📥 Installation

```bash
git clone https://github.com/Yashwenth27/GitMe.git
cd GitMe
pip install -r requirements.txt
```

## ▶️ Usage

1. Run the Streamlit app:

```bash
streamlit run main.py
```

2. Paste your GitHub repository URL.
3. Click **Generate README.md**
4. Copy or download your new README.
